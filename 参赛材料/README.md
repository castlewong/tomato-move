# TRAE AI 创造力大赛参赛材料

项目名称：动动番茄钟 Pomo

建议赛道：学习工作

一句话定位：用浏览器视觉 AI 验证用户在番茄钟结束后是否真的活动，让“休息”从提醒变成可验证的健康闭环。

## 文件说明

- `01-报名帖.md`：发布到“大赛报名专区”的创意提案正文。
- `02-初赛作品帖.md`：发布到“大赛初赛专区”的 Demo 作品正文。
- `03-演示讲稿.md`：录屏、现场介绍或评委快速体验时使用。
- `04-抖音发布文案.md`：抖音人气通道的视频标题、正文和口播。
- `05-提交检查表.md`：报名、初赛和传播材料的最终核对清单。

## 使用顺序

1. 先完成报名帖中的 `[待填写]` 内容，并将 `MovePomodoro.html` 作为附件上传。
2. 报名审核通过后，登录大赛官网确认领取报名权益。
3. 补齐初赛帖中的体验地址、截图、Session ID 和报名帖链接。
4. 发布初赛作品帖，并再次确认附件和链接可以正常访问。

## 官方要求摘要

- 报名帖正文不少于 100 字，需包含创意介绍、目标用户及痛点、价值与意义。
- 报名帖必须附带 TRAE 生成的创意产物 HTML 文件。
- 初赛作品需要可运行、可体验，支持体验链接或 HTML 文件。
- 初赛帖应清晰展示 TRAE 开发过程，建议至少提供 3 张关键步骤截图和 3 个关键任务 Session ID。
- 不要把本文档中的 `[待填写]` 占位符原样发布。
