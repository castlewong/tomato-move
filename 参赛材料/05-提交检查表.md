# TRAE AI 创造力大赛提交检查表

> 当前官方报名与初赛截止时间：2026 年 7 月 15 日 23:59（北京时间）。提交前再次查看官方公告，确认时间和要求没有变化。

## 一、报名帖

- [ ] 使用与 TRAE 中国版相同手机号注册的社区账号。
- [ ] 标签选择 `学习工作`。
- [ ] 标题包含赛道和作品名称。
- [ ] 正文不少于 100 字。
- [ ] 已包含创意介绍、目标用户及痛点、价值与意义。
- [ ] 已上传 `MovePomodoro.html`，不是只粘贴 GitHub 链接。
- [ ] HTML 附件可以正常下载和打开。
- [ ] 已删除文案里的所有 `[待填写]` 和“发布前删除”内容。
- [ ] 发帖后保存报名帖链接。
- [ ] 次日检查官网、短信或公示中的审核状态。
- [ ] 报名通过后登录 https://www.trae.cn/ai-creativity/result 确认领取权益。

## 二、初赛作品帖

- [ ] 报名已经审核通过。
- [ ] 在“大赛初赛专区”单独发布作品帖。
- [ ] 标签选择 `学习工作`。
- [ ] 清楚说明 Demo 是什么、面向谁、主要功能是什么。
- [ ] 给出推荐体验路径，让评委不必等待 25 分钟。
- [ ] 至少提供一种可运行体验方式：HTTPS 链接或 HTML 附件。
- [ ] 如果使用摄像头，在线体验地址是 HTTPS。
- [ ] 上传至少 3 张 TRAE 开发关键步骤截图。
- [ ] 填写至少 3 个关键任务 Session ID。
- [ ] 展示从创意、第一版、问题发现到重构优化的过程。
- [ ] 补充报名帖链接、GitHub 链接和演示视频链接。
- [ ] 所有链接在未登录或无缓存环境下也能访问。
- [ ] 删除所有 `[待填写]` 占位符。

## 三、Demo 自测

- [ ] 页面能在 Chrome/Edge/Safari 最新版本打开。
- [ ] 开始、暂停、重置、提前结束按钮正常。
- [ ] 提前结束后能显示有效休息弹层。
- [ ] “暂不休息”可以关闭弹层，但不会增加有效休息次数。
- [ ] 摄像头权限拒绝时有明确提示。
- [ ] 摄像头开启后，画面没有被提示层大面积遮挡。
- [ ] 头部、双肩和手肘入镜即可进入识别状态。
- [ ] 转头和手臂活动可以被识别。
- [ ] 静止时不会快速误判为完成。
- [ ] 完成后“有效休息”次数只增加一次。
- [ ] 手机尺寸下没有横向滚动和内容溢出。
- [ ] 刷新页面后当日数据仍然存在。
